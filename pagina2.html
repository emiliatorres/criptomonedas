Tipos de Bitcoin

  1. Criptomonedas de pago
    • Son monedas digitales diseñadas principalmente para funcionar como medio de intercambio. Su objetivo es ser utilizadas como dinero, permitiendo pagos rápidos y descentralizados sin necesidad de intermediarios.
  2. Criptomonedas de plataformas
    • Son monedas que sirven como combustible interno de una plataforma blockchain. Generalmente se usan para pagar comisiones, ejecutar contratos inteligentes y participar en la red.
  3. Stablecoins
    • Son criptomonedas cuyo valor está ligado (anclado) a un activo estable, como el dólar, el euro o incluso el oro. Están diseñadas para mantener una estabilidad de precio.
  4. tokens de utilidad
    • Son tokens que brindan acceso a funcionalidades o servicios específicos dentro de una plataforma. No son dinero en sí, sino una especie de "ficha" para interactuar dentro de un ecosistema cripto.
